# 2gis
# Анализ заведений Алматы по данным 2GIS

## Описание проекта
Анализ кафе, ресторанов и баров Алматы с использованием публичного API 2GIS.
Проект показывает какие категории заведений преобладают и на каких улицах их больше всего.

## Технологии
- Python
- Requests (парсинг API)
- Pandas (обработка данных)
- Matplotlib & Seaborn (визуализация)

## Что сделано
- Сбор данных через 2GIS API
- Очистка и структурирование данных
- Категоризация заведений по типу
- Визуализация топ улиц и категорий

## Основные выводы
- Самая популярная категория — **Кафе** (56%)
- Больше всего заведений на **проспекте Абая**
- Бары занимают второе место по популярности (20%)

## Как запустить
1. Открыть `2gis.ipynb` в Google Colab
2. Вставить свой API ключ от 2GIS
3. Запустить все ячейки по порядку
